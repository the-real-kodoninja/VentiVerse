## Setup

1.  **Install DFINITY SDK:**
    ```bash
    sh -ci "$(curl -fsSL [https://sdk.dfinity.org/install.sh](https://sdk.dfinity.org/install.sh))"
    ```
2.  **Start Local IC:**
    ```bash
    dfx start --background
    ```
3.  **Deploy Backend:**
    ```bash
    cd backend && dfx deploy
    ```
4.  **Install Frontend Dependencies:**
    ```bash
    cd frontend && npm install
    ```
5.  **Run Frontend:**
    ```bash
    npm run dev
    ```
